## 📌 Project Name

**KinKeeper**

## 📝 Description

KinKeeper is a modern web application designed to manage and interact with friends or contacts efficiently. It provides a clean UI and essential communication features like calling, messaging, and organizing contacts.

## ⚙️ Technologies Used

* React.js
* Tailwind CSS
* React Router
* Context API
* React Icons
* Vite

## ✨ Features

* 📇 View and manage friends list
* 📞 Call functionality UI
* 💬 Chat interface design
* 🎥 Video call option (UI)
* ⚡ Fast and responsive design
* 🔄 Dynamic data handling with custom hooks
* 🎯 Clean and user-friendly interface
